API 호출 문서(`requests.http`) 생성을 완료했습니다.

이 파일을 사용하려면:

1.  **Spring Boot 애플리케이션 실행:**
    프로젝트 루트 디렉토리에서 다음 명령어를 실행하여 애플리케이션을 시작합니다.
    ```bash
    ./gradlew bootRun
    ```
2.  **API 호출:**
    `http-client/requests.http` 파일을 VS Code (REST Client 확장 설치 필요) 또는 IntelliJ IDEA에서 연 후, `GET http://localhost:8080/hello` 라인 위에 표시되는 'Send Request' 버튼을 클릭하면 API가 호출되고 응답을 확인할 수 있습니다.
